Automation clase uno.

Consiste en realizar una breve prueba automatizada con Python y Selenium.

Pre-condiciones:

1- Tener instalada la última versión de python o a partir de la versión 3.8.0 (Ingresar en la terminal: python -versión)
2- Instalar Selenium - pip install Selenium
3- Instalar chromedriver según versión de google Chrome https://chromedriver.storage.googleapis.com/index.html . Para ver la versión de Chrome ir a Help - About Google Chrome
4- Consejo: Saber donde estará ubicado el ChromeDriver, en especial se recomienda una ruta de acceso fácil como C://DriverChrome/ChromeDriver 

Prueba automatizada:

URL a testear: https://demo.seleniumeasy.com/jquery-download-progress-bar-demo.html

1- Se ingresa a la URL
2- Se hace click en el botón descargar
3- Se espera a que se complete la descarga
4- Se cierra el chromedriver
